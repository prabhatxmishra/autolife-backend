## autolife-backend
Production-grade Spring Boot backend for managing tasks, reminders, subscriptions, and generating intelligent daily insights.

## 🚀 Features (Planned)
- Task management (CRUD)
- Reminder system (one-time & recurring)
- Subscription/bill tracking
- Smart insights APIs
- Scheduler for alerts

## 🛠 Tech Stack
- Java
- Spring Boot
- PostgreSQL
- JPA / Hibernate
